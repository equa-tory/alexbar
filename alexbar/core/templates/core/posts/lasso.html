Lasso

Lasso is a console program written on python that helps you download audio from YouTube very quickly and conveniently. You can download a single video or an entire playlist by choosing the format (3gp, aac, flv, m4a, mp3, mp4, ogg, wav, webm), location and range.

Lasso - это консольная программа написанная на питоне которая помогает очень быстро и удобно скачивать и аудио с ютуба. Вы можете скачивать единый ролик или целый плейлист выбирая формат (3gp, aac, flv, m4a, mp3, mp4, ogg, wav, webm), место и диапазон.

Installation:

Установка:

Clone/install Github Repository Клонируйте/скачайте Github Репозиторий

pip install -r requirements.txt

Linux

Arch Linux:

sudo pacman -S ffmpeg

Debian/Ubuntu:

sudo apt install ffmpeg

Windows

Needful utility for convertation ffmpeg (latest realese) [direct link] - export zip at any folder search for system variables and add to "Path" the path of ...\ffmpeg-version-essentials_build\bin restart your terminal

Нужна утилита для конвертации ffmpeg (latest realese) [прямая ссылка] - извлеките файлы в любую папку.

python main.py

Open the application "Changing system environment variables", "Environment Variables...", open "Path" in the top menu and create a link to ...\ffmpeg-version-essentials_build\bin

Откройте приложение "Изменение системных переменных среды", "Переменные среды...", в верхнем меню откройте "Path" и создайте ссылку на ...\ffmpeg-version-essentials_build\bin

Finally restart the terminal

В конце перезапустите терминал
